Les modules nécessaire

Afin de pouvoir essayer notre version il faut installer certains modules notamment tkinter, ainsi que pygame.

Installation des modules

Afin d'installer ces modules, veuillez ouvrir un terminal et écrire :
pip install tkinter
pip install pygame
pip install partial
Nous vous expliquerons quelques fonctions de notre code qui nous semblent primordial

Creation de la grille joueur

La variable plateau permet d'avoir une image de fond sur la grille.
La variable plateau_joueur permet de placer l'image et de l'afficher.
Les variables label_joueur,canva_grille_joueur permettent d'afficher la grille des joueurs.
La première boucle for permet donc de numéroté les cases.

Cela est identique pour l'ordinateur sauf qu'elle sera générée plus tard.

Creation des bateaux

La variable canva_all_boat permet d'avoir une liste de rectangles transparent ils seront utilisé pour avoir un déplacement conforme aux règles. Les images seront alors placé dessus.

Par la suite, on observe qu'on va importer une image de porte avion et l'ajouter en utilisant une fonction coords qui nous permet donc de récupérer la position du rectangle concerné, on va donc itérer cela à tous les images de bateaux.
